Name:
Sum of numbers from 0 to N

Description:
Given value N, compute the sum of the range 1..N and pretty print it in the format presented below:

> f 6
"0+1+2+3+4+5+6 = 21"

# If N is below 0:
> f -1
"-1<0"

# If N is zero:
> f 0
"0=0"
