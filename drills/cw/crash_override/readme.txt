Name:
Crash Override

Description:
Given two pre-defined hashes with a letter to a corresponding nickname mapping, FIRST_NAME and SURNAME, generate an alias by taking the first letter of the name and the last name passed into the method. If any of the letters is not alphabetical, please return a string: "Your name must start with a letter from A - Z."
