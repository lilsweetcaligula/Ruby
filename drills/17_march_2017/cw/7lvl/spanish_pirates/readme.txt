Name:
Spanish Pirates Off the Treasure Coast

Description:
Count the number of unit-arrays with a string "pirate" in them, and the number 
of unit-arrays with a string "pile_of_gold", and output the string in the format:

"count of pirates: #{count_of_pirates} and the count of gold piles: #{count_of_gold_piles}"
